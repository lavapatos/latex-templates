## Cómo usar

1. Navega al template que necesitas
2. Descarga o clona el repositorio
3. Copia la carpeta del template a tu proyecto
4. Edita el archivo `.tex` según tus necesidades

## Cómo contribuir

Ver [CONTRIBUTING.md](CONTRIBUTING.md) para instrucciones detalladas.

**Resumen rápido:**
1. Crea tu template en la carpeta correspondiente
2. Agrega un archivo `metadata.json`
3. Haz un Pull Request
