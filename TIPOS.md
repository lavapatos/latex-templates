# Tipos de Documento

Usa estos valores en el campo `template_type` del `metadata.json`.

| Tipo | Valor |
|------|-------|
| Afiche TICs | `afiche-tics` |
| Ayudantía | `ayudantia` |
| Control | `control` |
| Guía de apoyo | `guia-apoyo` |
| Informe de laboratorio | `lab-report` |
| Mesa de estudio | `mesa` |
| Presentación | `presentation` |
| Trabajo final | `tf` |
| Tutoría | `tutoria` |
